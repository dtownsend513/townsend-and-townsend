# MASTER WEBSITE TEMPLATE

Reusable Next.js + Tailwind + shadcn/ui website framework for rapid client deployments.

---

# Stack

- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui
- Lucide Icons

---

# Features

- Responsive Navbar
- Mobile Drawer Menu
- Hero Sections
- Features Sections
- Services Grid
- Stats Section
- Gallery Section
- Testimonials
- FAQ Accordion
- CTA Banner
- Contact Form
- Theme System
- Data System
- Image System
- Reusable Components
- Reusable Layouts

---

# Installation

```bash
npm install
```

---

# Run Development Server

```bash
npm run dev
```

Open:

```txt
http://localhost:3000
```

---

# Build Production

```bash
npm run build
```

---
